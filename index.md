**CCTG** (Claude Code Tmux Gateway) ties together **tmux + Claude Code + a chat gateway (Telegram or Discord)** on macOS, so you can **run and manage a per-project Claude Code chat bot** from your phone or any chat client.

- 🔌 **Connect Claude Code to Telegram / Discord bots** — one command, `cctg`.
- 🧩 Each project bot gets its own **isolated tmux session**, token, and state directory.
- 📱 Start tasks and check status **remotely from chat** — no terminal to babysit.
- 🔒 Safe defaults: a shared permission policy + deny safety net; never touches the global channel bot.
